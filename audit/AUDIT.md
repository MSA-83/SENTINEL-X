## Security Patches Applied

### 1. SSRF Vulnerability (CWE-918) - ✅ PATCHED
- **File**: `agents/base_agent.py:79-97`
- **Issue**: Unvalidated entity_id in URL construction
- **Fix**: Added RFC-1918 blocklist regex, input validation

### 2. Hardcoded API Keys (CWE-798) - ✅ PATCHED
- **Files**: `agents/base_agent.py`, `agents/classifier_agent.py`
- **Issue**: Hardcoded "your-free-groq-key" and "free-groq-key"
- **Fix**: Changed to `os.environ.get("GROQ_API_KEY", "")`

### 3. Prompt Injection (CWE-79) - ✅ PATCHED
- **Files**: `agents/base_agent.py:122-151`, `agents/classifier_agent.py:168-185`
- **Issue**: Unsanitized entity_data in LLM prompts
- **Fix**: Added INJECTION_PATTERNS filter with re.sub

### 4. SQL Injection (CWE-89) - ✅ VERIFIED SAFE
- **File**: `convex/entities.ts`
- **Status**: Convex query system is safe (uses parameterized queries internally)

### 5. Data Poisoning (CWE-74) - ✅ PATCHED
- **File**: `agents/ml_dataset_pipeline.py`
- **Issue**: Unvalidated training data ingestion
- **Fix**: Quality threshold (0.7), SHA-256 provenance hash
